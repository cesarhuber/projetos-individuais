Descrição
É uma versão virtual do famoso jogo de cartas Blackjack. Aqui você joga contra o computador (a mesa). O objetivo é chegar o mais próximo possível da pontuação de 21, no entanto, sem estourar esta pontuação. Se terminar a rodada com mais que 21 pontos, você automaticamente perde.

Tecnologias utilizadas
Para realizar este projeto, foi utilizada a tecnologia Javascript, incorporada ao HTML para rodar no seu navegador.
